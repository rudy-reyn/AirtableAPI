# Airtable Datatypes

## Overview
airtable.datatypes is a submodule that maps Airtable field types to corresponding python objects.

The following subclasses are currently supporting:

 Python Class | Airtable Data Type 
---|---
 Autonumber | Autonumber
 Count      | Count
 Currency   | Currency
 Number     | Number
 Percent    | Percent
 Rating     | Percent
---|---
